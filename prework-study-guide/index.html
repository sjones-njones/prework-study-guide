Gitbash CLI commands

  • CLI: command line interface-allows functionality for sites through texts
  • GUI: graphical user interface
  • pwd: print working directory
  • ls: list of files in working directory
  • cd name: change directory
  • cd ..: moves back one level
  • mkdir name: make directory
  • touch name.txt: creates text file
  • start name: opens file

Git

  • Git commands do not = bash commands and always start with GIT
  • Git tracks changes locally
  • git add -A: saves staged files
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
